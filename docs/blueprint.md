# **App Name**: CourtControl AI

## Core Features:

- Club & Identity Management: Comprehensive interface for registering a sports club, managing administrative users, and uploading branding assets like logos.
- Multi-Category Tournament Wizard: A step-by-step configuration tool to set up tournaments with varied formats such as single elimination or round robin across multiple courts.
- Participant Preference Engine: Handles registrations for teams and individuals, collecting vital player data including gender mix, t-shirt sizes, and sponsor pack preferences.
- Visual Scheduling & Drag-and-Drop: Interactive timeline for managing match schedules with the ability to manually adjust court assignments and times via drag-and-drop interactions.
- OR-Tools Smart Scheduler: A mathematical optimization tool that builds non-conflicting schedules based on court availability and player rest times via a Cloud Function.
- QR Telegram Check-In: System to generate printable QR codes for venue entrances that map physical arrival to player Telegram handles for automated match notifications.
- Digital Score Console: Real-time referee scoring interface designed for zero-friction mobile entry without physical paper trails.
- Interactive Score Verification: Secure referee score entry system that triggers mandatory player approval requests via Telegram notifications to ensure result accuracy.
- Live Arena Dashboard: Optimized Next.js display interface for high-resolution salon/TV screens showing real-time court status and upcoming matches.
- Global Player Stats Hub: A centralized dashboard for players to monitor their performance metrics and generate high-impact visuals for social media sharing.

## Style Guidelines:

- Primary color: Electric Violet (#8B5CF6) for high-energy tech branding.
- Background color: Deep Space Charcoal (#0F172A) to minimize glare during outdoor court use.
- Accent color: Radiant Cyan (#22D3EE) for status indicators and active action buttons.
- Headline font: 'Space Grotesk' for a technical, modern sports aesthetic.
- Body copy font: 'Inter' for high legibility in dense tournament brackets and data tables.
- Responsive 'Bento Box' grid system for administrative dashboards to allow glanceable court monitoring.
- Smooth layout transitions and pulse effects on active courts during live tournament play.
